Technology Stack

  • Mongodb

    A NoSQL databasing program, that is an open-source, cross-platform, document-oriented database.

  • Express

    A minimalistic and flexible framework, providing a thin layer of fundamental web application features for Nodejs.

  • Angular

    An open-source Javascript web application framework based on type-script and designed by Google.

  • Nodejs

    An open-source cross-platform Javascript runtime environment built on Chrome's V8 Javascript engine, that allows the execution of Javascript on the server.

  • Nginx

    An open-source versatile web server, capable of reverse-proxing, high performance load balancing, mail proxing & HTTP caching.

  • Ubuntu

    A free open-source GNU/Linux operating system based on Debian, created by Canonical.
